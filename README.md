# c1-2
# 🍲 '맛있는 육수' 브랜드 아이덴티티 정의

---

## 1. 브랜드 아이덴티티

### 🎯 타겟 (Target Audience)

| 구분 | 내용 |
|------|------|
| **1차 타겟** | 30대-70대 주부 / 요리에 관심 있는 1인 가구 (20~30대) |
| **2차 타겟** | 간편하지만 제대로 된 맛을 원하는 바쁜 직장인 |
| **공통 심리** | *"손이 많이 가는 건 싫지만, 맛은 절대 포기 못해"* |

---

### 🎨 톤앤매너 (Tone & Manner)

> **"따뜻하고 신뢰감 있는, 그러나 세련된"**

- **감성** → 집밥의 따뜻함, 엄마의 손맛 연상
- **언어** → 친근하고 쉬운 말투 (과장 없이 진정성 있게)
- **비주얼** → 김이 모락모락 나는 국물, 자연재료 클로즈업, 따뜻한 색감(앰버/크림톤)
- **금지 요소** → 지나친 과학적 설명, 차갑고 딱딱한 기업 이미지

---

### ⭐ 차별점 / USP (Unique Selling Point)

```
"육수를 '우리는' 것과 똑같은 맛을,
코인 하나로 10초 만에"
```

| 경쟁사 대비 | 맛있는 육수 |
|------------|------------|
| 분말 육수 → 인공적인 맛 | 코인 압축 → 진한 천연 재료 그대로 |
| 액상 육수 → 보관 불편 | 코인 형태 → 휴대·보관 간편 |
| 직접 우리기 → 시간 소요 | 10초 용해 → 즉시 완성 |

**핵심 USP 한 줄 요약**
> *"타협 없는 맛, 타협 없는 편리함"*

## 2. 광고 캠페인 정의

### 📌 광고 목적
> **브랜드 인지 (Awareness) + 구매 전환 (Conversion)**
> → 코인육수라는 카테고리 자체가 생소한 소비자에게
> **"이런 게 있었어?"** 를 느끼게 한 뒤 바로 구매 욕구로 연결

---

### 💬 핵심 메시지 (Key Message) — 1문장

> ## *"육수 우릴 시간은 없어도, 맛을 포기할 이유는 없다"*

| 요소 | 의도 |
|------|------|
| **"시간은 없어도"** | 타겟의 현실 공감 유도 |
| **"맛을 포기할 이유는 없다"** | USP(편리함+맛) 동시 전달 |
| **전체 톤** | 강요 없이, 타겟 스스로 고개 끄덕이게 만드는 공감형 메시지 |

---

## 🎬 10초 숏폼 스토리보드 — 씬별 상세 설명

---

### 🔴 S1 | 0~3초 — PAIN POINT (공감 + 감정이입)
> **"육수 우릴 시간이 어딨어..."**

- **화면** : 바쁘게 움직이는 주인공 → 냄비 앞에서 시계 보며 한숨
- **카메라** : 시계 클로즈업 → 주인공 얼굴 미디엄샷 (빠른 컷 전환)
- **자막** : *"육수 우리는 데 2시간..."*
- **사운드** : 보글보글 끓는 소리 + 바쁜 배경음 → 음악 멈춤 (한숨)

---

### 🟡 S2 | 3~6초 — SOLUTION (반전 포인트)
> **"코인 하나면 돼!"**

- **화면** : 코인육수가 물에 '퐁당' 떨어지며 빠르게 녹는 장면
- **카메라** : 익스트림 클로즈업 (물속 코인 용해 슬로우모션)
- **자막** : *"코인 하나, 10초면 끝"*
- **사운드** : 경쾌한 '퐁당' 효과음 + 업비트 음악 시작

---

### 🟡 S3 | 6~8초 — RESULT (욕구 자극)
> **"10초 만에 완성!"**

- **화면** : 황금빛 육수가 완성되는 탑뷰 샷, 김이 모락모락
- **카메라** : 탑다운 샷 → 측면 클로즈업
- **자막** : *"진하고 깊은 맛 그대로"*
- **사운드** : "완성!" 효과음 + 음악 클라이맥스

---

### 🟣 S4 | 8~10초 — CTA (행동 유도)
> **"맛을 포기할 이유가 없다"**

- **화면** : 제품 패키지 풀샷 + '맛있는 육수' 로고
- **카메라** : 제품 정면 고정샷
- **자막** : *"맛있는 육수"*
- **사운드** : 브랜드 징글 + 내레이션 *"맛있는 육수 - 지금 바로 만나보세요"*

---

## ⏱ 타임라인 요약

| 구간 | 씬 | 역할 | 핵심 감정 |
|------|-----|------|----------|

| 8~10s | S4 | 구매 행동 유도 | *"사야겠다"* |
