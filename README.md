

## 1. 브랜드 아이덴티티

### 🎯 타겟 (Target Audience)

| 구분 | 내용 |
|------|------|
| **1차 타겟** | 30대-40대 주부, 워킹맘 / 요리에 관심 있는 1인 가구 (20~30대) |
| **2차 타겟** | 간편하지만 제대로 된 맛을 원하는 바쁜 직장인 |
| **공통 심리** | *"손이 많이 가는 건 싫지만, 맛은 절대 포기 못해"* |

---

### 🎨 톤앤매너 (Tone & Manner)

> **"따뜻하고 신뢰감 있는, 그러나 세련된"**

- **감성** → 집밥의 따뜻함, 엄마의 손맛 연상
- **언어** → 친근하고 쉬운 말투 (과장 없이 진정성 있게)
- **비주얼** → 김이 모락모락 나는 국물, 자연재료 클로즈업, 따뜻한 색감(앰버/크림톤)
- **금지 요소** → 지나친 과학적 설명, 차갑고 딱딱한 기업 이미지

---

### ⭐ 차별점 / USP (Unique Selling Point)

```
"육수를 '우리는' 것과 똑같은 맛을,
코인 하나로 10초 만에"
```

| 경쟁사 대비 | 맛있는 육수 |
|------------|------------|
| 분말 육수 → 인공적인 맛 | 코인 압축 → 진한 천연 재료 그대로 |
| 액상 육수 → 보관 불편 | 코인 형태 → 휴대·보관 간편 |
| 직접 우리기 → 시간 소요 | 10초 용해 → 즉시 완성 |

**핵심 USP 한 줄 요약**
> *"타협 없는 맛, 타협 없는 편리함"*

## 2. 광고 캠페인 정의

### 📌 광고 목적
> **브랜드 인지 (Awareness) + 구매 전환 (Conversion)**
> → 코인육수라는 카테고리 자체가 생소한 소비자에게
> **"이런 게 있었어?"** 를 느끼게 한 뒤 바로 구매 욕구로 연결

---

### 💬 핵심 메시지 (Key Message) — 1문장

> ## *"육수 우릴 시간은 없어도, 맛을 포기할 이유는 없다"*

| 요소 | 의도 |
|------|------|
| **"시간은 없어도"** | 타겟의 현실 공감 유도 |
| **"맛을 포기할 이유는 없다"** | USP(편리함+맛) 동시 전달 |
| **전체 톤** | 강요 없이, 타겟 스스로 고개 끄덕이게 만드는 공감형 메시지 |

---

## 🎬 8초 숏폼 스토리보드 — 씬별 상세 설명

---

### 🟡 S1 | 0~2초 — PAIN POINT (공감 + 감정이입)
> **"육수 우릴 시간이 어딨어..."**

- **화면** : 주인공(앞치마를 입은 30-40대 여성) 뚝배기 앞에서 벽시계 보며 한숨
- **카메라** : 벽시계 클로즈업 → (피곤해 보이는)주인공 얼굴 미디엄샷 (빠른 컷 전환)
- **자막** : 흰색 굵은 글씨로 *"육수 우리는 데 2시간..."* 글자가 타이핑되는 애니메이션.
- **사운드** : 째깍째깍 시계소리 + 보글보글 끓는 소리  → 소리 멈추고 한숨

---

### 🟡 S2 | 3~4초 — SOLUTION (반전 포인트)
> **"코인 하나면 돼!"**

- **화면** : 베이지색 동전 모양의 코인육수가 뚝배기속 된장찌개에 '퐁당' 떨어지며 빠르게 녹는 장면
- **카메라** : 익스트림 클로즈업 (찌개속 코인육수 용해 슬로우모션)
- **자막** : 흰색 굵은 글씨로 *"코인 하나, 10초면 끝"* 글자가 부드럽게 페이드인되는 애니메이션.
- **사운드** : 경쾌한 '퐁당' 효과음 + 업비트 음악 시작

---

### 🟡변경전-> S3 | 5~6초 — RESULT (욕구 자극)
> **"10초 만에 완성!"**

- **화면** : 황금빛 육수가 완성되는 탑뷰 샷, 김이 모락모락
- **카메라** : 탑다운 샷 → 측면 클로즈업
- **자막** : *"진하고 깊은 맛 그대로"*
- **사운드** : "완성!" 효과음 + 음악 클라이맥스


### 🟡변경후-> S3 | 5~6초 — RESULT (욕구 자극)
> **"10초 만에 완성!"**

- **화면** : 주인공이 뚝배기 안의 국물을 국자로 떠서 마시는 동작 애니메이션. 
               국물을 입에 댄 순간, 주인공의 표정이 감탄과 황홀함으로 변화.

주인공 표정 연출 상세 :
-눈이 동그랗게 커지며 놀라는 표정
-이후 눈이 반달 모양으로 휘어지며 미소
-볼에 분홍빛 홍조가 번지는 애니메이션 효과
-머리카락이 살랑 흔들리며 감동받은 느낌 연출
-말풍선 or 반짝이는 별 이펙트가 캐릭터 주변에 등장

- **카메라** : 미디엄샷 → 주인공 클로즈업
- **자막** : 흰색 굵은 글씨로 *"진하고 깊은 맛 그대로"* 글자가 부드럽게 페이드인되는 애니메이션.
- **사운드** : 업비트 경쾌한 BGM 계속(육수 찰랑이는 소리 추가) → 내레이션 "진하고 깊은 맛 그대로" 밝고 경쾌한 목소리로

프롬프트 수정 이유: 단순히 완성된 육수만 보여주는 것이 아닌 그 코인육수 제품을 경험한 사용자의 반응을 보여줌으로서, 시청자의 몰입감이 높아지고 맛을 간접적으로 상상하기 쉬울 것이라고 판단했다. 

---

### 🟡 S4 | 7~8초 — CTA (행동 유도)
> **"맛을 포기할 이유가 없다"**

- **화면** : 제품 패키지 풀샷 + '맛있는 육수' 로고
- **카메라** : 제품 정면 고정샷
- **자막** : 흰색 굵은 글씨로 *"맛있는 육수 — 지금 바로 만나보세요"* 화면 하단에 고정
- **사운드** : 브랜드 징글 + 내레이션 *"맛있는 육수"*

---

##  필수 포함
├── 제품 패키지 정면 사진 (고해상도 / 누끼 작업 권장)
├── 브랜드 로고 파일 (PNG 투명배경)
├── 패키지 중앙에 코인육수와 국물요리 사진 포함
├── 브랜드 컬러 코드
│    ├── 메인 : 크림 화이트 #FFF8EE
│    └── 텍스트 : 딥 브라운 #3E1F00
└── 참고 레퍼런스 이미지 (백설 1분링)


---

##  제작 시 주의사항

1. S1과 S4의 색감 차이를 극명하게 유지할 것
   → 대비가 클수록 "반전" 효과 강해짐

2. 실사 제품샷(S4)과 애니메이션(S1~S3)의
   이질감을 줄이기 위해
   → 제품 주변 스파클/빛망울 애니메이션 이펙트 필수

3. 전체 8초 안에 모든 씬이 끊김 없이 연결되도록
   각 씬 전환 효과(0.2~0.3초)를 시간에 포함시킬 것

4. 자막은 항상 화면의 안전 영역(Safe Zone) 안에 배치
   → 상하 각 10% 영역은 자막 배치 금지

## 생성된 미디어 소스를 해상도/비율/톤앤매너 관점에서 정리하고, 하나의 광고 영상으로 통합하는 흐름을 설명할 수 있다.

생성된 미디어 소스 정리

멀티모달 콘텐츠 제작에서는 텍스트, 이미지, 비디오, 오디오를 각각 생성한 뒤, 최종 결과물에 맞게 규격과 스타일을 통일해야 한다.

|   항목   |	  고려 사항	  |   예시   |
| --------| ---------------| ---------|
| **해상도(Resolution)** |	최종 영상의 해상도에 맞게 통일하여 화질 저하를 방지	| Full HD(1920×1080), 4K(3840×2160) |
| **비율(Aspect Ratio)** |  플랫폼에 맞는 화면 비율을 적용  |	16:9(유튜브), 9:16(숏폼), 1:1(SNS) |
| **톤앤매너(Tone & Manner)** |	색감, 분위기, 글꼴, 음악, 말투 등을 일관성 있게 유지	|  밝고 경쾌한 광고, 고급스럽고 차분한 브랜드 이미지 |

[광고 영상으로 통합하는 흐름]

1단계. 기획
광고 목적과 타깃 고객을 설정한다. 핵심 메시지와 스토리보드를 작성한다.

2단계. 미디어 생성

각 생성형 AI 도구를 활용해 필요한 콘텐츠를 제작한다.

텍스트 생성 → 광고 카피, 대본 작성
이미지 생성 → 배경, 제품 이미지, 일러스트 제작
비디오 생성 → 제품 소개 장면, 애니메이션 생성
오디오 생성 → 내레이션, 배경음악(BGM), 효과음(SE) 제작

3단계. 미디어 소스 정리

생성된 자료를 하나의 기준으로 맞춘다.

모든 이미지를 동일한 해상도로 변환
영상의 프레임 크기와 화면 비율 통일
브랜드 컬러와 색감 보정
글꼴과 자막 디자인 통일
음량과 음질을 일정하게 조정

4단계. 영상 편집 및 통합

편집 프로그램을 이용하여 하나의 광고 영상으로 완성한다.
이 과정에서 자막, 전환 효과, 배경음악, 효과음을 추가하여 완성도를 높인다.

5단계. 최종 검수 및 출력

최종 영상을 확인하여 품질을 점검한다.

해상도 확인
화면 비율 확인
자막 오류 수정
음량 균형 확인
색감 및 톤앤매너 일관성 확인

완성된 영상은 목적에 맞는 형식으로 출력한다.

유튜브 → 16:9, Full HD 또는 4K
인스타그램 릴스·틱톡 → 9:16
SNS 게시물 → 1:1 또는 4:5
